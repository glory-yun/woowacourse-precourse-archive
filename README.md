# 📂 Woowa Course Precourse Archive

**우아한 테크코스 프리코스** 과정을 통해 학습하고 구현한 과제들을 모아둔 아카이브입니다.
각 주차별 미션의 요구사항을 분석하고, 클린 코드 원칙과 객체 지향 설계를 적용하며 성장했던 과정을 기록했습니다.

## 🗂️ Directory Structure

이 저장소는 프리코스 주차별 미션을 별도 폴더로 관리합니다. 각 폴더명을 클릭하면 해당 프로젝트의 상세 코드를 확인할 수 있습니다.

| 폴더명 | 주차 | 주제 | 주요 학습 내용 |
| :--- | :---: | :--- | :--- |
| **[`precourse_1`](./precourse_1)** | 1주차 | **문자열 덧셈 계산기** | • 기능 단위 분리 및 커밋 컨벤션 준수<br>• 사용자 입력 처리 및 구분자 파싱 로직 구현 |
| **[`precourse_2`](./precourse_2)** | 2주차 | **자동차 경주** | • MVC 패턴 적용 시도<br>• 테스트 코드(Jest) 작성 및 리팩토링 경험 |
| **[`precourse_3`](./precourse_3)** | 3주차 | **로또** | • 도메인 로직과 UI 로직의 분리<br>• 예외 처리(Exception Handling) 강화 및 클래스 분리 |
| **[`woowa-memoir`](./woowa-memoir)** | 오픈 미션 | **회고 및 오픈 미션** | • 프리코스 과정 전체 회고<br>• Java 기반 오픈 미션 수행 및 학습 기록 |
